# ReactNative-Quarkus

A React Native application for uploading Images and videos (similar to Instagram).

## Tech Stack

- Quarkus API (Reactive programming for User Auth and Transactional logic)
- Nativewind (CSS)
- Expo (Routing & DevEx)
- Postgres (Database)
